# Admin Guide - How to Upload Content for Students

## Quick Start

### 1. Login as Admin
- Go to `/admin/login` or use the regular login page with admin credentials
- Default admin credentials (change after first login):
  - Email: `admin@example.com`
  - Password: `admin123`

### 2. Create Content Structure

The content hierarchy is: **Class → Chapter → Topic**

#### Step 1: Create a Class
1. Go to **Admin Dashboard** → Click **"Create Class"** or navigate to **Classes** section
2. Select class type:
   - Class 5
   - Class 6
   - Class 7
   - Class 8
   - Class 9
   - Class 10
   - SSC Prep
   - General Knowledge (GK)
3. Enter class name (e.g., "Class 5 Mathematics")
4. Add optional description
5. Click **"Create Class"**

#### Step 2: Create Chapters
1. Go to **Chapters** section or click **"Manage"** on a class card
2. Click **"Create Chapter"** or **"Add Chapter"**
3. Select the class
4. Enter chapter title (e.g., "Introduction to Numbers")
5. Add optional description
6. Set order number (for sequencing)
7. Click **"Create Chapter"**

#### Step 3: Create Topics with Content
1. Go to **Topics** section or click **"Manage Topics"** on a chapter
2. Click **"Create Topic"** or **"Add Topic"**
3. Select the chapter
4. Enter topic title (e.g., "Basic Addition")
5. Set order number
6. **Write content using the rich text editor:**
   - Format text (bold, italic, underline)
   - Add headings
   - Create lists (ordered/unordered)
   - Add links and images
   - Align text
7. Click **"Create Topic"**

### 3. Content Management

#### View All Content
- **Classes**: `/admin/classes` - See all classes and their chapter counts
- **Chapters**: `/admin/chapters` - View all chapters across all classes
- **Topics**: `/admin/topics` - See all topics with their content

#### Edit Content
- Click **"Edit"** button on any topic to modify content
- Use the rich text editor to update content
- Changes are saved immediately

#### Organize Content
- Use **order** numbers to control the sequence of chapters and topics
- Lower numbers appear first
- You can reorder by editing the order field

### 4. Monitor Student Activity

#### View Statistics
- Dashboard shows counts of:
  - Classes
  - Chapters
  - Topics
  - Students
  - Community Posts

#### Community Moderation
- Go to **Community** section to view all student posts and replies
- Monitor discussions and ensure appropriate content

## Content Creation Workflow

```
1. Create Class (e.g., "Class 5")
   ↓
2. Create Chapters (e.g., "Mathematics", "Science")
   ↓
3. Create Topics (e.g., "Addition", "Subtraction")
   ↓
4. Students can now access content!
```

## Tips for Effective Content Creation

1. **Organize by Class**: Create separate classes for different grade levels
2. **Use Clear Titles**: Make chapter and topic titles descriptive
3. **Rich Formatting**: Use the editor to make content visually appealing
   - Use headings for sections
   - Add bullet points for lists
   - Include images where helpful
4. **Order Matters**: Set logical order numbers for sequential learning
5. **Descriptions Help**: Add descriptions to help students understand what they'll learn
